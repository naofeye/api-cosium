# 19 - User stories et critères d’acceptation

## Exemples structurants
- En tant qu''assistante, je veux voir immédiatement les pièces manquantes d''un dossier.
- En tant que responsable, je veux rapprocher un paiement bancaire avec plusieurs factures.
- En tant que chargé marketing, je veux segmenter les clients éligibles à une campagne SMS.
